# Iris Flower Classification using Machine Learning

This project demonstrates how to classify iris flowers into different species using machine learning.  
It uses the classic **Iris dataset**, one of the most popular datasets for beginners in data science and machine learning.

---

## Project Overview

The Iris dataset contains measurements of three different species of iris flowers â€” *setosa*, *versicolor*, and *virginica*.  
Using these features, we train models to predict the species based on sepal and petal length and width.

In this project, we:
- Load and explore the Iris dataset.
- Perform data visualization and feature analysis.
- Train and evaluate multiple ML models.
- Compare model performance.

---

## Algorithms Used
- Logistic Regression  
- Support Vector Machine (SVM)  
- K-Nearest Neighbors (KNN)  
- Decision Tree  
- Random Forest  

---

## Dataset

The dataset is included in the `sklearn.datasets` library.

**Features:**  
- Sepal length  
- Sepal width  
- Petal length  
- Petal width  

**Target:**  
- Iris Setosa  
- Iris Versicolor  
- Iris Virginica
